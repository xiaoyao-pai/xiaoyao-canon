# Skill 经验库搭建实战

> 来源：逍遥派经验沉淀 | 脱敏版

## Skill 是什么

你的专业经验，打包成 AI 能理解的操作手册。不是文档——文档写了放那落灰，Skill 加载进 AI 就直接生效。

## 三种 Skill 类型

| 类型 | 作用 | 举例 |
|------|------|------|
| **流程 Skill** | 做事的方法 | 头脑风暴 → 编写计划 → 执行 → 验证 |
| **质量 Skill** | 兜底的机制 | 完成前验证、代码审查、TDD |
| **领域 Skill** | 专业知识 | 部署架构、项目经验、踩坑记录 |

## Skill 文件规范

```markdown
---
name: 技能名称
description: 一句话描述
triggers:
  - 触发词1
  - 触发词2
---

# 技能内容

## 快速概览
（表格或简短文字）

## 执行步骤
（具体操作流程）

## 核心规则
（必须遵守的 2-3 条）
```

## 设计原则

- SKILL.md ≤ 100 行，详情放 modules/ 和 references/
- 触发词要覆盖用户可能的表达方式
- 渐进式披露：L1 概览 → L2 模块 → L3 详情
- 一个 Skill = Prompt + 输入输出规范 + 自检逻辑 + 版本管理

## 从个人到团队

1. 解决问题 → 跑通一个流程
2. 固化为 Skill → 下次一句话调用
3. 团队共享 → 任何人都能复用你的经验
