# Token 成本控制指南

> 来源：逍遥派经验沉淀 | 脱敏版

## 核心原则

省钱不是少用 AI，是别在不需要好模型的地方花冤枉钱。

## 三招省钱

| 方法 | 做法 | 效果 |
|------|------|------|
| **规则精简** | 只保留高频使用的规则，其余降级为 Skill 按需加载 | Token 降 52% |
| **按需加载** | Skill 不全量注入，特定场景才触发 | 不浪费上下文 |
| **模型分级** | 简单任务用轻量模型，复杂任务用旗舰模型 | 性价比最优 |

## 免费资源利用

- 硅基流动：免费额度，支持 DeepSeek/Qwen 等主流模型
- GitHub Pages：静态项目零成本部署
- 腾讯云轻量服务器：最低约 50 元/月

## Agent 配置精简经验

- AGENT.md 从 310 行砍到 63 行，system prompt 缩减 15%，响应反而更稳定
- Skill 从 41 个减到 14 个——真正在用的不到一半
- 重复强调同一规则是浪费——说一次 AI 就记住了
- 禁用不需要的工具（tools.deny），减少 system prompt 体积
