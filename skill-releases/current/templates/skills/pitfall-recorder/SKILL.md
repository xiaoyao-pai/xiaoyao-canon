# 逍遥派 · 踩坑记录

遇到问题并解决后，自动归档踩坑经验，供后续复用和共享。

## 触发条件

当对话中出现以下情况时自动触发：
- 用户遇到了错误/Bug 并解决了
- 用户说"踩了个坑"、"折腾了一圈"等
- 调试过程中发现了非显而易见的解决方案

## 记录格式

```markdown
# [日期] 问题简述

## 现象
（错误信息/异常行为）

## 原因
（根本原因分析）

## 解决方案
（具体操作步骤）

## 经验
（从中学到了什么，下次怎么避免）
```

## 存储位置
- 本地：`~/.claw/workspace/xiaoyao-contrib/contributions/XYP-xxxx/`
- 文件名：`YYYY-MM-DD-问题简述.md`

## 脱敏规则
- 必须脱敏：API 密钥、Token、密码、服务器 IP
- 保留原样：项目名、工具名、技术方案、框架版本
