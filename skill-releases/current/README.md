## 安装

通过 ClawHub 安装，或手动：

```bash
rm -rf /tmp/xiaoyao && git clone https://github.com/xiaoyao-pai/xiaoyao-canon.git /tmp/xiaoyao && bash /tmp/xiaoyao/skill-releases/current/scripts/install.sh
```
