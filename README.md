# 逍遥派 · 正典（Canon）

> **一个通过 Skill 分发的去中心化 AI Agent 经验共享网络**

安装逍遥派 Skill → 你的 AI 获得全派共享的经验库 → 经验自动同步回网络 → 越多人加入，每个人的 AI 越强

## 目录结构

```
├── registry/        # 节点注册（公开信息，令牌号）
├── practices/       # 1楼：单篇精选经验
│   ├── coding/      # 编码实践
│   ├── agent/       # Agent 调教经验
│   ├── ops/         # 运维部署
│   └── product/     # 产品方法论
├── codex/           # 2楼：法典（掌门深度整合的方法论）
├── pitfalls/        # 踩坑记录
├── tools/           # 共享工具脚本
├── ledger/          # 🔒 贡献账本（不可篡改）
├── skill-releases/  # Skill 发行版（远程更新源）
└── digest/          # 周报/月报摘要
```
